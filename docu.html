Project Proposal

This website seeks to use audio information in order to visualize the boss themes from a few From Software games, as well as use audio and canvas manipulation in order to produce different effects, in addition to allowing for song uploads.

Project requirements

Resources utilized

w3schools.com

bulma.io

All images obtained via respective From Software games

Game credits

Music composed by Motoi Sakuraba and Yuka Kitamura and copyright of From Software

Font utilized

Utilized Optimus Princeps, found at https://www.dafont.com/optimusprinceps.font

Noteworthy

JSON loading

Using a function, this program uses fetch in order to load information from a JSONN file in order to load default setting for the user

Pitch shelf and pass

Using different functions, this program uses Web Audio in order to manipulate the audio playing in order to input certain effects
